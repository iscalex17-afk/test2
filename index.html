Întrebări frecvente

Pot opri animația de fundal?

Da. Apasă „Pauză animație” din hero sau folosește setarea de sistem Reduce Motion.

Cum înlocuiesc produsele?

Editează titlurile, descrierile și imaginile din secțiunea Produse. Recomandat 1200×750px, format .webp.

Pot folosi Netlify / Vercel?

Da, este un site static. Formularul din secțiunea Contact este compatibil Netlify.

Suport pentru brand personalizat?

Schimbă variabilele CSS --gold și --gold-2 sau fonturile pentru a adapta tema.
