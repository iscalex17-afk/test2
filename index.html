Întrebări frecvente

Pot opri animația de fundal?

Da. Apasă „Pauză animație” din hero sau folosește setarea de sistem Reduce Motion. Site-ul detectează preferința și ajustează automat.

Este compatibil cu Netlify / Vercel?

Da. Este un site static cu un formular compatibil Netlify. Poți face deploy pe orice platformă de hosting static.

Consumă multă baterie?

Nu. Redăm liniile în batch și reducem rata de cadre atunci când fereastra e inactivă. Se folosește requestAnimationFrame și throttling adaptiv.

Pot schimba culorile glow-ului?

Da. Editează variabilele CSS --accent și --accent-2. Modificările sunt aplicate instant.

Oferiți factură proformă?

Da, pentru planurile Pro și Team. Completează formularul și menționează „proformă” în mesaj.

Am nevoie de JS separat?

Nu. Această versiune are CSS și JS inline pentru simplitate. Poți extrage fișierele ulterior.
